# B.Home - Site de Construção de Casas Personalizadas

Este é um site para uma empresa fictícia especializada em construção de casas personalizadas de acordo com a vontade do cliente. O site foi desenvolvido com HTML, CSS e JavaScript puros, sendo totalmente compatível com hospedagem no GitHub Pages.

## Características

- Design responsivo para todos os dispositivos
- Animações de scroll e interações dinâmicas
- Galeria de projetos com filtro por categoria
- Ferramenta de personalização interativa para visualizar diferentes estilos de casas
- Slider de antes/depois para mostrar transformações
- Carrossel de depoimentos de clientes
- Formulário de contato funcional (simulado)

## Estrutura do Site

1. **Header**: Logo, navegação e menu mobile
2. **Hero Section**: Banner principal com chamada para ação
3. **Serviços**: Cards destacando os principais serviços
4. **Portfólio**: Galeria de projetos filtráveis
5. **Personalizador**: Ferramenta interativa para visualizar opções de casas
6. **Antes/Depois**: Slider comparativo
7. **Depoimentos**: Carrossel de testemunhos de clientes
8. **Contato**: Formulário e informações de contato
9. **Footer**: Links úteis e informações da empresa

## Tecnologias Utilizadas

- HTML5
- CSS3 (com variáveis CSS e flexbox/grid)
- JavaScript (vanilla)
- Font Awesome para ícones
- Google Fonts para tipografia

## Como Executar

Basta abrir o arquivo `index.html` em qualquer navegador moderno.

## Hospedagem

Este site pode ser facilmente hospedado no GitHub Pages:

1. Crie um repositório no GitHub
2. Faça upload de todos os arquivos do projeto
3. Ative o GitHub Pages nas configurações do repositório

## Personalização

Para personalizar este site para uso real:

1. Substitua as imagens de placeholder por imagens reais de projetos
2. Atualize as informações de contato
3. Modifique as cores no arquivo CSS (variáveis no início do arquivo)
4. Conecte o formulário de contato a um backend real
